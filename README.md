## 📊 Project Overview
DeepScout is a next-generation sports analytics and football scouting dashboard built to transform raw match performance data into actionable recruitment intelligence. By integrating comprehensive statistical baselines with position-specific algorithmic scoring, DeepScout filters, ranks, and visualizes the top performers across 9 major European leagues.

### 🧬 Core Features
* **Multi-League Data Integration:** Seamless harmonization of comprehensive player profile metrics and dynamic performance data.
* **Algorithmic Spectrum Scoring:** Position-specific weighting models for **Physical**, **Speed & Transition**, **Creative**, and **Finishing** profiles.
* **Custom Dynamic Visualization:** Interactive DNA Radars and progressive matrix bars for visual performance tracking.
* **Scouting Quality Control:** Built-in adajustments including league difficulty coefficients and a strict **minimum 10-match baseline** to eliminate statistical noise.
